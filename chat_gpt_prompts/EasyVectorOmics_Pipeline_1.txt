EasyVectorOmics Pipeline:

Input: TPM (Salmon)

Normalization:
- gene-wise centering (division by gene's standard deviation over all samples)
- quantile normalization over all samples
- log transformation using: log( x + 1 )
- if stress conditions: log( stress + 1 ) - log( control + 1 )

Coordinate calculation:
- mean of tissue replicates, or
- if stress: mean of stress minus mean of control

Gene-Family cloud (solar system identification):
for each gene family, do:
- find orthologs
- compute ortholog centroid (mean ortholog expression)
- compute all distances of paralogs to the ortholog centroid
- compute relative divergence index (RDI) of these distances by dividing
  the distances with the family max

Compute empirical distribution of RDIs:
- find outliers using RDI>Q3+1.5×IQR
- compute P-Value using 
empirical_p <- function(x, RDI_vector) {
  sum(RDI_vector >= x) / length(RDI_vector)
}

alternatively: rank based 
empirical_p_rank <- function(x, RDI_vector) {
  rank <- sum(RDI_vector >= x)
  pval <- rank / length(RDI_vector)
  return(pval)
}

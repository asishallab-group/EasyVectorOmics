public pure subroutine build_kd_index(points, num_dimensions, num_points, kd_indices, dimension_order, workspace, value_buffer, permutation, left_stack, right_stack, recursion_stack, ierr)

Build a k-d tree index using a stack-based, non-recursive approach. Initialize kd_indices to 1:num_points (original indices) Choose split dimension by cycling through dimension_order Find median index Partition kd_indices(left_idx:right_idx) by points(current_dim, kd_indices(:)) Push right and left intervals onto stack

subroutine normalization_pipeline_r(n_genes, n_tissues, input_matrix, buf_stddev, buf_quant, buf_avg, buf_log, temp_col, rank_means, perm, stack_left, stack_right, max_stack, group_s, group_c, n_grps, ierr)

Uses

    • tox_normalization

R/Fortran wrapper for normalization pipeline. Provides an interface for R (.Fortran) and Fortran code to call the normalization pipeline routine. Performs: std dev normalization, quantile normalization, replicate averaging, log2(x+1) transformation. Final result is in buf_log. If fold change is needed, call calc_fchange separately.

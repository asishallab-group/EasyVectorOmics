public subroutine loess_alloc(x, y, span, degree, fitted_values, mode, n_iters, ierr)

Uses

    • iso_fortran_env

Wrapper subroutine for LOESS fitting (plain or robust). This subroutine selects between plain and robust LOESS fitting based on the mode. It dynamically allocates the required arrays and computes workspace sizes.

Parameters: - mode: Specifies the type of LOESS fitting to perform. - 0: Plain LOESS fitting. This mode performs a single pass of LOESS fitting without any additional weighting or iterations. It is suitable for datasets without significant outliers. - 1: Robust LOESS fitting. This mode applies bisquare reweighting over multiple iterations to reduce the influence of outliers. The number of iterations is controlled by the n_iters parameter.

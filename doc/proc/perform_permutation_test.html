public subroutine perform_permutation_test(trajectories, n_factors, n_samples, n_timepoints, factor_idx, dependent_idx, sample_idx, mode, n_permutations, local_contributions, total_contributions, temp_factor, temp_dependent, ierr, random_seed)

For a given factor-dependent pair, this subroutine calculates the contributions by taking the same dependent but from a random different sample.

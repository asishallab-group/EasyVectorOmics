public subroutine normalization_pipeline_alloc(n_genes, n_replicates, expr, log_transformed_expr, reps_per_tissue, n_tissues, span, degree, use_quantile, ierr)

Complete normalization pipeline for gene expression data. Performs: std dev normalization, quantile normalization, replicate averaging, log2(x+1) transformation. Final result is in log_transformed_expr. If fold change is needed, call calc_fchange separately.

public pure subroutine log2_transformation(n_genes, n_tissues, expr, transformed_expr, ierr)

Author
Vivian Bass

Apply log2(x + 1) transformation to each element of the input matrix. This subroutine performs element-wise log2(x + 1) transformation on a matrix flattened in column-major order. The log2 is computed via: log(x + 1) / log(2), which is numerically equivalent and avoids the non-portable log2 intrinsic for compatibility with WebAssembly (WASM).
